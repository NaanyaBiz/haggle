# Code of Conduct

This project follows the **Contributor Covenant 2.1**. The full text
lives at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## In short

- Be respectful. Disagreement is fine; harassment is not.
- Assume good faith. Reviewers and contributors are volunteers.
- Keep discussion technical and on-topic.
- Confidentiality matters: do not paste real account numbers, real meter
  reads, or real PKCE callback URLs into public issues or PRs.

## Reporting

Conduct issues, like security issues, are handled privately by the
maintainer:

- **Email**: `security@naanya.biz`

We will acknowledge reports within 5 business days and act in accordance
with the Contributor Covenant's enforcement guidelines.

## Scope

This Code applies in all project spaces — issues, PRs, discussions,
commit messages, and any communication channel where a contributor is
representing the project.
